⚙️ MIDI Routing

🔵 Bluetooth Foot Controller Setup (MidiBerry + loopMIDI)

  1. Install loopMIDI → create a virtual port (e.g. loopMIDI Port)
  2. Install MidiBerry → pair your Bluetooth foot controller
  3. In MidiBerry, route BT MIDI output → loopMIDI Port
  4. Open Chrome, allow MIDI access when prompted
  5. Click Refresh Ports below → select loopMIDI Port from dropdown
  6. Press a pedal — the green activity light should flash
Note: Web MIDI requires Chrome/Edge. Firefox is not supported.
