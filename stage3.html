🔵 MidiBerry + loopMIDI Setup

  1. Install loopMIDI → create virtual port: loopMIDI Port
  2. Install MidiBerry → pair BT foot controller → route to loopMIDI Port
  3. Use Chrome or Edge — Firefox does not support Web MIDI
  4. Click Refresh Ports → select loopMIDI Port → Save
  5. Step on a pedal — the green ● flashes when signal arrives
